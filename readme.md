GraphSearch
===============

Operations with graphs:
 - breadth-first search
 - depth-first search

### Installation

```sh
$ cd /path/to/htdocs
$ git clone https://github.com/volmir/graph-search.git
$ composer install
```

### Run

```sh
$ php index.php
```

### Tests

```sh
$ phpunit --bootstrap vendor/autoload.php tests
```
